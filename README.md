# Event Agency Website Platform

Production website built for an event agency, combining marketing, ticket sales, and lead generation into a single system.

## Overview

The platform serves as:
- Public-facing website for events and services
- Ticket sales entry point via Qtickets
- Lead capture system with direct email delivery
- Analytics-driven conversion tracking

## Key Features

- Ticket purchase flow with external integration (Qtickets)
- Lead capture forms with corporate email delivery
- Conversion tracking via Yandex Metrica
- CMS-driven content management (Directus)
- Interactive UI (parallax, smooth scrolling, video sections)

## Architecture

- Frontend: Next.js
- CMS: Directus (custom data model)
- Analytics: Yandex Metrica
- Ticketing: Qtickets
- Email: corporate inbox integration

## Proof of Functionality

The system is live and used in production:

- Real ticket interactions tracked via analytics
- Active lead submissions from users
- Event promotion and audience engagement

See `/docs` for screenshots and case summary.

## Execution

The entire system was designed and delivered independently in ~3 weeks under real business constraints.
